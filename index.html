Node.js

Czym jest Node.js ?


sdsdsNode.js to otwarte, wieloplatformowe środowisko uruchomieniowe dla aplikacji napisanych w języku JavaScript.


Dzięki wykorzystaniu silnika V8, pozwala tworzyć aplikacje uruchamiane poza przeglądarką internetową.

Zalety

  • szybkość
  • możliwość obsłużenia dużej liczby zapytań na sekundę
  • nieblokujące operacje I/O
  • małe zużycie pamięci
  • ogromna liczba modułów stworzonych przez społeczność

Zastosowanie

  • aplikacje sieciowe typu real-time
  • aplikacje nie wymagające dużych zasobów CPU
  • Tworzenie API typu REST
  • Aplikacje typu CLI
  • aplikacje desktopowe z użyciem HTML, CSS, JS, ELECTRON

Historia Node.js

Platforma ta powstała w 2009 roku, a jej twórcą jest Ryan Dahl

2010

pierwsza wersja package managera npm

2011

Microsoft wraz z firmą Joyent wypuszcza pierwszą wersję Node.js dla Windows

2012

Ryan Dahl odsuwa się od projektu i przekazuje go w ręce twórców npm

2014

powstaje bliźniaczy projekt o nazwie io.js, który zakłada szybszy cykl wydawania nowych wersji, w tym za każdym razem, gdy Google wyda nową wersję silnika V8

zmienia się również lider projektu Node.js

2015

powstaje Node.js Foundation i połączenie wersji 0.12 Node.js z wersją 3.3 io.js i wydanie nowej wersji Node.js oznaczonej numerem 4.0
